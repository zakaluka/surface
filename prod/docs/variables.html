Variables

Change these variables to suit your websites needs. All of the below can be found in the variables file unless otherwise stated.

Code

Typography

You can change the fonts used and size of type elements.

Colors

To quickly change the color scheme of a website, change the $primary, $secondary, $accent and $body-background colors

To change or add colors, you can find them in the _colors file in the _imports folder.

Grid

You can set the grid to support as many columns as you like by altering the $grid-columns variable.

Decorative

There are a few things you can change within the decorative section. You can alter the box shadows, or remove them by setting them all to none. You can change the timing function for all transitions. You can change or remove the border radius too.

Spacing

The spacing units can be changed.

Media Queries

You can change the breakpoints of $medium, $small and $tiny that the grid and margin system respond to.
